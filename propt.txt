# PROMPT: Build a bilingual static site (RO/EN) for PFA + self-accounting + e-bike rentals for couriers

**Goal:** Create a modern, mobile-first, static marketing website that targets food delivery riders (Bolt Food, Glovo, Wolt, etc.) in Bucharest/Romania (but inclusive for non-Romanian speakers like Nepali/Sri Lankan/Indian riders). The site promotes:

1. Creating a PFA from scratch
2. Teaching riders to handle basic self-accounting
3. Renting electric bikes (best option for Bolt couriers vs. car/scooter)

## Tech constraints

* **Plain HTML + JS** only (no frameworks).
* **Tailwind CSS** via CDN.
* Single-page site (can have internal sections + modal).
* **Bilingual:** Romanian (default) and English.
* Language switcher on the top-right; persist choice in `localStorage`.
* All strings stored in a simple `i18n` object in a separate `lang.js` (or inline script)—no server.
* Clean, accessible, WCAG-friendly.
* Lightweight (good Lighthouse scores).

## Brand & style

* Clean, sleek, high-contrast, courier-friendly.
* Icons for clarity (SVG inline).
* Rounded cards, soft shadows, generous spacing, big tap targets.
* Sticky header with CTA, sticky bottom CTA on mobile.
* Neutral color palette + one accent color for CTAs.
